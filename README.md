* Challenge Staking
* NFT Staking
# TALE heroes
The Hero of Tales Of Chain is BEP-721 tokens. Each hero is represented by their own smart contract.

# Tests
To run the tests, install the dependencies:
```console
$ npm install
```
And run the tests:
```console
$ truffle test
```
